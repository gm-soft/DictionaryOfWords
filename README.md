# Приложение словарик

## Задание 
###	Реализовать программу которая будет хранить языки, слова и переводы слов

## Задачи
* языки могут изменяться
* слова могут изменяться
* сделать фильтрацию
* сделать постраничный вывод данных
* реализовать на ASP.Net Core mvc 2.2 + mySQL
